### 8. Instructor Do: Defining Object Store Data with Indexes (5 mins)

* Use the prompts and talking points below to demonstrate the following key point(s):

  * âœ” Object stores are schema-less and have no native search capability.

  * âœ” We create indexes on object store "columns" so we can query.

  * âœ” We use the `createIndex(indexName, keyPath)` function to create indexes, it takes two arguments.

  * âœ” The `indexName` is what you use to access the index when querying.

  * âœ” The `keyPath` is the actual name of the "column."

  ```js
  request.onupgradeneeded = ({ target }) => {
    const db = target.result;
    const objectStore = db.createObjectStore("toDoList");
    objectStore.createIndex("timestamp", "timestamp");
  };
  ```
