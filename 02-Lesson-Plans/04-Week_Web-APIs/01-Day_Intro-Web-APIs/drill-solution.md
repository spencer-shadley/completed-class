# Write a function that takes in two numbers and outputs the max (the greater of the two numbers)

```js
var max = function(num1, num2) {
  if (num1 > num2) {
    return num1;
  }
  else {
    return num2;
  }
}
```

## Bonus

```js
var max = function(num1, num2) {
    return num1 > num2 ? num1 : num2;
}
```
