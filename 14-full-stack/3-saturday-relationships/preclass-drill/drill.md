# Pre-class Drill

Determine if a collection of any number of inputs contain a given sub-word

---

Input

sub-word: `car`

inputs: `racecar`

Output

`true`

---

Input

sub-word: `car`

inputs: `racecar`, `haircare`, `carbon`

Output

`true`

---

Input

sub-word: `car`

inputs: `pizza`

Output

`false`
